Help

The managed switch console lets you interact with your network configuration using commands similar to Juniper or Cisco devices. You can view, edit, and commit configuration changes.

  • show – Display current configuration or state.
  • set – Modify configuration data.
  • delete – Remove configuration entries.
  • commit – Apply your changes.
  • quit – Leave the CLI session and saves the command history.
  • ? – Display help.

See the Clixon CLI Quickstart for more details.
